# RNBF Project Page (GitHub Pages)

## Publish
1. Create a GitHub repo (e.g., `rnbf`)
2. Upload everything in this folder to the repo root
3. Repo → Settings → Pages → Deploy from branch → `main` → `/ (root)`

Your site will be:
`https://YOUR_USERNAME.github.io/rnbf/`

## Replace placeholders (minimal work)
- Teaser image: `media/teaser.png`
- Pipeline figure: `media/pipeline.png`
- PDF: `media/paper.pdf`
- Videos:
  - `media/videos/sim_demo.mp4`
  - `media/videos/real_demo.mp4`

## Update links
In `index.html`, replace the `Code` button `href="#"` with your GitHub code URL.
